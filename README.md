
# Laravel Absensi Guru

## Cara Install

1. Ekstrak project ke:
   C:\laragon\www\absensi-guru

2. Buka terminal:
   cd C:\laragon\www\absensi-guru

3. Install dependency:
   composer install

4. Copy env:
   copy .env.example .env

5. Generate key:
   php artisan key:generate

6. Migrasi database:
   php artisan migrate

7. Jalankan server:
   php artisan serve
